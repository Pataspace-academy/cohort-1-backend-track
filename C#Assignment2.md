# C# Operators Assignment

## 1. Objective
Build a console application in C#.

## Requirements: 
Have a .NET SDK installed in your machine.
VS Code with C# Dev Kit extension.

## Tasks
1. Using the different arithmetic operations, build a simple calculator where a user enters both a value and a symbol i.e addition, subtraction, product, division sign and gets a result of the operation. NOTE: Its a user input calculator and also make use of any control structure taught, either if, if-else, while or do-while loop.

2. Using the While Loop structure, write a block of code to output a list of 10 numbers from 0-10.

3. Using the do - while loop structure, write a block of code to allow user enter a number and print a list of numbers. NOTE: User input.. and list of numbers should maximize at 5.
