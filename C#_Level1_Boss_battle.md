# C# Level 1 Boss Battle - Interactive Console Application

## 1. Objective

Build a fully functional menu-driven console application in C# that combines everything you have learned across Assignments 1, 2 and 3 into one real project.

## Requirements

- .NET SDK installed on your machine
- VS Code with C# Dev Kit extension

---

## The Mission

Build **one** of the following applications:

- **Option A:** Student Grade Calculator
- **Option B:** Simple Quiz Game

You choose. Both must meet the same requirements below.

---

## Tasks

### Task 1 - Main Menu

Build a menu that displays on startup and accepts user input to navigate options.

Example:

```
===========================
   PataSpace Grade System
===========================
1. Enter Student Grades
2. View Results
3. Exit
Enter your choice:
```

---

### Task 2 - Control Flow

Your application must use:

- At least one `if / else if / else` block
- At least one `switch` statement
- At least one loop (`for`, `while`, or `do-while`)

---

### Task 3 - Custom Methods

Your application must have **at least 3 custom methods**. Each method must have:

- A clear, descriptive name
- Parameters where necessary
- A return type (not everything should be `void`)

Example methods you might write:

```csharp
static void DisplayMenu()
static string GetGrade(int marks)
static void ShowResults(string name, string grade)
```

---

### Task 4 - User Input & Validation

Your application must:

- Accept real user input throughout
- Handle empty or invalid input gracefully — do not let the app crash on bad input

---

### Task 5 - Clean Code

Your submission will be evaluated on:

- Meaningful variable and method names
- Comments explaining your logic
- No dead or unused code left in the file
