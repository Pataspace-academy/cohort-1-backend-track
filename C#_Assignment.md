# C# User Input Assignment Brief

## 1. Objective
Build a console application in C#.

## Requirements: 
Have a .NET SDK installed in your machine.
VS Code with C# Dev Kit extension.

## Tasks
1. Write a code that prompts the user for their name and age, and prints Hello, {name} and {age}.

2. Modify the program to reect empty input and ask again until a valid name is entered.

3. Create a simple user input calculator that can perform addition, subtraction, division and multiplication.

## Submission
1. Initialize a Git repository:
- git init
- git add .
- git commit -m "Initial commit"

2. Create a new repo on Github and push your code:
- git remote add origin  <your-repo-url>
- git push -u origin main

3. Submit the Github repository link on the specified location indicated.
